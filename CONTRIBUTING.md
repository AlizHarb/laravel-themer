# Contributing to Laravel Themer

Thank you for considering contributing to Laravel Themer!

## Branching Strategy

- **main**: The latest stable version.
- **develop**: The branch for the next release.

## Coding Style

We follow the [PSR-2](https://www.php-fig.org/psr/psr-2/) coding standard and the [PSR-4](https://www.php-fig.org/psr/psr-4/) autoloading standard.

Please verify code style with Laravel Pint:

```bash
./vendor/bin/pint
```

## Testing

You can run the tests with:

```bash
./vendor/bin/pest
```
