# Laravel Boost

Laravel Themer v1.4.0 ships Laravel Boost package resources:

```text
resources/boost/guidelines/core.blade.php
resources/boost/skills/laravel-themer-development/SKILL.md
```

After installing Laravel Boost in an application, run:

```bash
php artisan boost:install
```

If Laravel Themer is installed after Boost, rediscover package resources:

```bash
php artisan boost:update --discover
```

Boost-aware agents will understand theme manifests, inheritance, preview URLs, diagnostics, refresh workflows, and design tokens.
