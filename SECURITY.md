# Security Policy

If you discover a security vulnerability in Laravel Themer, please email Ali Harb at [harbzali@gmail.com](mailto:harbzali@gmail.com).

Please do not disclose security issues publicly until a fix is available.

Supported versions:

| Version | Supported |
| --- | --- |
| 1.4.x | Yes |
| 1.3.x | Yes |
| < 1.3 | No |
